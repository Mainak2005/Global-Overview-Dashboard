🌍 Global Sales Overview Dashboard – Power BI
📌 Project Overview

This project is an interactive Power BI Dashboard designed to analyze and visualize global sales performance across different regions, categories, and shipping modes. The dashboard provides key business insights such as total sales, order trends, regional performance, and sub-category analysis to support data-driven decision-making.

📊 Dashboard Preview

🚀 Key Features
KPI Cards
Total Sales
Total Order Count
Average Sales
Sales Trend Analysis
Monthly sales performance visualization
Year-wise trend filtering using slicer
Regional Performance
Sales comparison across regions
Combined Sales & Order Count analysis
Sub-Category Analysis
Best-performing product sub-categories
Funnel-style visualization for contribution analysis
Shipping Insights
Average shipping days by shipping mode
Interactive Filters
Dynamic year range slicer for trend analysis

🛠️ Tools & Technologies Used
Power BI Desktop
Data Visualization
DAX (Data Analysis Expressions)
Data Cleaning & Transformation using Power Query

📈 Insights Derived
The West region generated the highest sales revenue.
Phones and Chairs were among the top-selling sub-categories.
Sales showed strong growth during the final months of the year.
Standard Class shipping mode had the highest average delivery time.
